1) Посмотреть где я - pwd
2) Создать папку - mkdir git-project
3) Зайти в папку - cd git-project
4) Создать 3 папки mkdir - folder folder1 folder2
5) Зайти в любоую папку - cd folder
6) Создать 5 файлов (3 txt, 2 json) - touch text.txt text1.txt text2.txt 535.json 526.json
7) Создать 3 папки - mkdir lesson1 lesson2 lesson3
8. Вывести список содержимого папки - ls -la
9) + Открыть любой txt файл - cat text.txt , либо vi text.txt
10) + написать туда что-нибудь, любой текст. - echo "test" > test.txt , либо i, написать текст, нажать Enter 
11) + сохранить и выйти. - Esc, :wq
12) Выйти из папки на уровень выше - cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку. - mv C:/Users/Username/git-project/folder/text.txt text1.txt C:/Users/Username/git-project/folder/lesson1
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. - cp text.txt text1.txt C:/Users/Username/Folder1
15) Найти файл по имени - find name FileName
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает. - tail -f text.txt
17) вывести несколько первых строк из текстового файла - head -n 4 text.txt
18) вывести несколько последних строк из текстового файла tail -n 2 text.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает. - less text.txt
20) вывести дату и время - date
